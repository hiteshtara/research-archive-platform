# Protocol Research Area Reconciliation

Run `sql/verify/protocol_research_area_reconciliation.sql` after loading.

The report covers source/archive row counts, unique source identifiers,
resolved, missing and ambiguous tuple parents, source-ID differences,
resolved-parent mismatches, missing research-area codes, and archive orphans.

Promotion requires reviewed focused Oracle evidence, zero missing or
ambiguous tuple parents, zero archive orphans, zero resolved-parent
mismatches, and no unexplained source/archive count difference.
